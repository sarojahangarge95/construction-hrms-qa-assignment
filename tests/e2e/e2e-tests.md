# E2E Test Suite

## Test 1: Employee Onboarding
- Create a new employee.
- Verify employee appears in employee list.
- Verify attendance tracking is activated.

## Test 2: Profile Update
- Update employee designation and salary.
- Verify changes are saved.
- Verify updated salary appears in next payslip.

## Test 3: Salary and Payslip Flow
- Enter salary details.
- Generate payslip.
- Verify salary calculations are correct.

## Test 4: Employee Exit
- Deactivate employee.
- Verify employee is removed from active list.
- Verify payroll processing stops.

# Bug: Login allows submission with empty username

## Severity
Medium

## Justification
Users may receive unclear validation messages and fail to log in correctly.

## Steps to Reproduce
1. Open the login page.
2. Leave the username field empty.
3. Enter a password.
4. Click Login.

## Expected Result
The system should display a validation message such as "Username is required".

## Actual Result
The system allows submission or displays a generic error message.

## Impact
Users cannot understand why login failed, resulting in poor user experience.

## Root Cause
Input validation may be missing on the client side.

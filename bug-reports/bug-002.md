# Bug: Employee search is case-sensitive

## Severity
Low

## Justification
Users may not find employees easily.

## Steps to Reproduce
1. Search for an employee using lowercase letters.
2. Search for the same employee using uppercase letters.

## Expected Result
Search should return matching employees regardless of letter case.

## Actual Result
Search results differ based on capitalization.

## Impact
HR users may struggle to locate employee records.

## Root Cause
Search implementation is case-sensitive.
